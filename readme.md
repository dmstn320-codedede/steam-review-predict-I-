# 🎮 Steam Review 기반 신뢰도 추천 시스템(BETA)

## 📌 1. 프로젝트 개요

게임 선택의 폭이 넓어지면서 사용자들은 수많은 정보 속에서 선택의 어려움을 겪고 있다.
특히 Steam은 사용자 리뷰 기반 평점 시스템을 제공하지만, 단순 긍정 비율만으로는 신뢰도를 충분히 반영하기 어렵다.

본 프로젝트는 리뷰 수를 반영한 신뢰도 기반 추천 모델을 설계하고,
장르 필터링 기능을 포함한 프로토타입 추천 시스템을 구현하는 것을 목표로 한다.


## 🎯 2. 문제 정의

Steam 평점은 다음과 같은 한계를 가진다.

리뷰 100개 95%와
리뷰 100,000개 93%는
같은 의미를 가지는가?

단순 긍정 비율만으로는 리뷰 수에 따른 신뢰도 차이를 반영하기 어렵다.

따라서 본 프로젝트는
신뢰도를 반영한 추천 점수 모델을 설계한다.


## 📊 3. 데이터 구성

기존 Steam 리뷰 데이터 약 2,100만 건

최신 수집 리뷰 데이터 병합

최근 3개월 데이터 필터링 적용

사용 컬럼:

app_name

recommended

timestamp_created



## 🧠 4. 모델 설계 과정
### 🔹 Model A — Recent Only

최근 3개월 긍정 비율만을 기준으로 순위 산출

한계:

리뷰 수 차이를 반영하지 못함

### 🔹 Model B — Log Weighted

긍정 비율 × log(리뷰 수)

장점:

리뷰 수에 따른 가중치 반영

한계:

리뷰 수가 적은 게임의 극단적 평점 보정 부족

### 🔹 Model C — Bayesian Hybrid (최종 선택)

베이지안 보정 적용

리뷰 수가 적은 경우 전체 평균에 가깝게 조정

리뷰 수가 많을수록 실제 평점에 가까워짐

최종 점수:

Final Score = Bayesian Score × log(전체 리뷰 수)

## 🎮 5. 추천 시스템 구조
✔ 베이지안 기반 신뢰도 점수 계산
✔ 장르 매핑 (Top50 기준)
✔ 복수 장르 선택 가능
✔ 장르 일치 수에 따른 가산점 적용
✔ TOP5 추천 + 시각화
✔ 결과 설명 자동 생성


## 🖥 6. 실행 방법
recommend_service(["RPG", "Action"])

FPS / RPG / Survival / Simulation / Roguelike
Indie / Visual Novel / Action / Strategy
Horror / Multiplayer / Adventure / Puzzle

📎 9. 프로젝트 의의

단순 평점 기반 추천이 아닌
